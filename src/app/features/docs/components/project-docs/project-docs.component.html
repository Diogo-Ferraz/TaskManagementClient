Task Management Platform

A Jira-style product showcase focused on clean architecture, secure auth, and production-minded Angular UX. This page explains the purpose, technical direction, and architecture decisions behind the platform.

Angular + PrimeNG OpenIddict + PKCE SignalR Real-time

Build a practical collaboration workspace where teams can plan, execute, and track work using boards, role-aware capabilities, and a live activity feed.

The frontend intentionally demonstrates maintainability and UI quality while proving real backend integration, not a disconnected mock-only demo.

The backend is built as a reusable template with Vertical Slice Architecture and OpenIddict-based auth. The SPA maps directly onto that backend contract so adding features remains predictable.

Error handling follows RFC7807 Problem Details, while key screens include resilient loading, empty, and degraded states for local UI development and demos.
