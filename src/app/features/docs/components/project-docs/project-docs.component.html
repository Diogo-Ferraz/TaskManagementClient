Task Management Platform

A Jira-style product showcase focused on clean architecture, secure auth, and production-minded Angular UX. This page explains the purpose, technical direction, and architecture decisions behind the platform.

Angular + PrimeNG OpenIddict + PKCE SignalR Real-time
Primary Goal Build a practical collaboration workspace where teams can plan, execute, and track work using boards, role-aware capabilities, and a live activity feed.
Frontend Intent The SPA demonstrates maintainability and UI quality while proving real backend integration, not a disconnected mock-only demo.
Backend Contract Alignment The backend is built as a reusable Vertical Slice template with OpenIddict auth. The SPA maps directly to that contract for predictable feature delivery.
Resilience Strategy Error handling follows RFC7807 Problem Details and key screens include loading, empty, and degraded states for local development and demos.
